# ddns-scripts_namesilo
适用于 OpenWRT/LEDE 自带DDNS客户端的 namesilo 更新脚本

依赖: ddns-scripts wget openssl-util

建议安装: luci-app-ddns


安装后在DDNS服务提供商一栏多出一个 namesilo.com。
